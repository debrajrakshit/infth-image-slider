This plugin display continious scrolling images

== Description ==

You can add images and display them in a infinite scrolling loop. Good for display client logos, portfolios, etc.

== Installation ==

Process 1:
Go to your WordPress admin `dashboard`. Then under `Plugins` menu click `Add New`. Click the `Upload Plugin` button and select the `plugin.zip` file and click `Install Now`. After installation complete click `Activate`.

Process 2:
Extract the `plugin.zip` archive and copy the files into your WordPress root > `/wp-content/plugins/` directory and paste files. Then go to your WordPress `dashboard > Plugins > Installed Plugins > Infinite Image Scroll` and click `Activate`.

You can display infinite image scroll using either one of the following:
1. Add this shortcode to display the slider into a page: `[infinite-image-scroll]`
2. Add this php code to your template: `<?php do_action('plugin_name_hook'); ?>`
